Теги таблицы

  1. ячейка <td> текст </td>
  2. ячейка-заголовок <th> текст </th>
  3. строка <tr> <th> или <td> </tr>
  4. <tbody> tr > td </tbody>
  5. пояснение 
     <caption>
	<h4> пояснение для таблицы</h4>
</caption>
  6. таблица <table> все теги таблицы </table>

Первичность строк / tr

Ячейка (th / td) всегда должна быть внутри строки tr

Расчет числа ячеек таблицы

  • столбцы: td / th число ячеек в каждом столбце одинаково с остальными столбцами таблицы
  • строки: tr число ячеек в каждой строке одинаково с остальными строками таблицы

CSS: border-collapse

Для таблиц определены две совершенно разные модели border

  • separate
    • каждая ячейка таблицы расположена на некотором расстоянии от других ячеек
    • рамки ячеек не сливаются друг с другом
    • нельзя задать border для строк, групп строк, столбцов и групп столбцов

CSS: border-collapse

Для таблиц определены две совершенно разные модели border

  • collapse
    • между border вокруг таблицы и ее внешними ячейками никогда не может быть пробела
    • можно задать border для таблицы, ячеек, строк, группам строк, столбцам и группам столбцов
    • при слиянии рамки между ячейками центрируются
    • половина ширины рамки таблицы включена в ширину таблицы
    • две или более рамки сливаются

CSS: border-collapse

separate
table border-collapse:separate;
table border-spacing:5px 1em;
table border:4px solid gold;
td border:4px solid green;

Пустые ячейки

Существует возможность скрыть оформление пустой ячейки в модели border-collapse: separate

empty-cells: hide; скрыть

empty-cells: hide

text text

Если все ячейки строки пусты и значение empty-cells для всех равно hide, то вся строка интерпретируется так, как будто для элемента строки задано display: none.

Размеры ячеек

  • height для всех ячеек одной строки / row будет одинаковой
  • height одной строки / row может отличаться от высоты другой
  • width для всех ячеек одного столбца будет одинаковой
  • width для различных столбцов может отличаться

Фиксированная и автоматическая ширина ячеек

table-layout:

  • fixed
    • любая ячейка с заданной шириной отличной от auto определит ширину столбца
    • если таблица оказывается шире суммарной ширины всех ее столбцов, разница делится на число столбцов и добавляется к каждому из них
  • auto пытается вместить ширину ячеек в заданную ширину таблицы

Выравнивание внутри ячейки

  • width используем text-align
  • height используем vertical-align
    • text-align: center center center left left left right right right justify justify justify
    vertical-align: top bottom middle baseline
